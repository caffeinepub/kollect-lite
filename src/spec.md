# Specification

## Summary
**Goal:** Refine the customer summary layout, limit activity timeline comments, and add visual containers to documents.

**Planned changes:**
- Limit activity timeline to display maximum two comments at a time
- Move phone number and phone icon inline with contract ID in customer summary
- Swap Customer ID to top left and Contract ID to bottom right in customer summary
- Enclose each document in a visual container spanning from document icon to preview icon

**User-visible outcome:** Users will see a cleaner customer summary layout with reorganized fields, a less cluttered activity timeline showing two comments, and better-defined document rows with visual containers.
